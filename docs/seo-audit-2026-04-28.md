## Critical Issues (Fix Immediately)

### C1 — No XML Sitemap
**Impact:** Googlebot has no discovery map for the site. The `@astrojs/sitemap` integration is not installed. With only 5–6 pages, Googlebot will likely find them via crawl, but for a new-ish static site with sparse backlinks, submitting a sitemap to Google Search Console is essential.

**Fix:**
```bash
npm install @astrojs/sitemap
```

`astro.config.mjs` — add the integration:
```js
import sitemap from '@astrojs/sitemap';

export default defineConfig({
  site: 'https://www.a1paralegal.com',
  output: 'static',
  integrations: [react(), sitemap()],
});
```
Then submit `https://www.a1paralegal.com/sitemap-index.xml` in Google Search Console.

---

### C2 — No robots.txt
**Impact:** No crawl guidance for any crawler. Not having one is not penalized, but it is a best practice gap and needed to reference the sitemap.

**Fix:** Create `public/robots.txt`:
```
User-agent: *
Allow: /
Disallow: /intake/

Sitemap: https://www.a1paralegal.com/sitemap-index.xml
```
> Note: `/intake/` should be disallowed — it is an internal operational form, not a public SEO page.

### C4 — Homepage H1 Is the Brand Name, Not a Keyword
**Impact:** The current `<h1>A1 Quality Paralegal</h1>` wastes the most important on-page keyword signal. The H1 should describe what the business *does* and *where*.

**Fix in** `src/pages/index.astro:11`:
```html
<!-- Before -->
<h1>A1 Quality Paralegal</h1>

<!-- After -->
<h1>Legal Document Preparation in Grants Pass, Oregon</h1>
```
Move the business name to the tagline element above it — it is already displayed prominently via the logo.

---

### C5 — No Open Graph or Social Meta Tags
**Impact:** Every page is invisible to social platforms (Facebook, LinkedIn, iMessage link previews). Anyone sharing the URL sees blank or garbage data. For a local business, word-of-mouth shares on Facebook/Nextdoor are a real traffic source.

**Fix in** `src/layouts/Layout.astro` `<head>`:
```html
<meta property="og:type" content="website" />
<meta property="og:title" content={`${title} | A1 Quality Paralegal`} />
<meta property="og:description" content={description} />
<meta property="og:url" content={`https://www.a1paralegal.com${pathname}`} />
<meta property="og:image" content="https://www.a1paralegal.com/images/og-image.jpg" />
<meta property="og:site_name" content="A1 Quality Paralegal" />
<meta name="twitter:card" content="summary_large_image" />
```

Also requires creating an OG image (1200×630px) — see Medium issues.

---

### C6 — No Canonical Tags
**Impact:** While Cloudflare enforces HTTPS, canonical tags prevent any potential duplicate content issues (e.g., with/without trailing slash, `www` vs non-`www`).

**Fix in** `src/layouts/Layout.astro` `<head>`:
```html
<link rel="canonical" href={`https://www.a1paralegal.com${pathname}`} />
```

**Impact:** Google's local knowledge panel pulls hours from GBP and from structured data. No hours on-site means a missed rich result opportunity. Users also need to know when to call.

### H5 — No About / Story Page — E-E-A-T Gap
**Impact:** A1 Quality Paralegal is a YMYL-adjacent business (legal services). Google's Quality Rater Guidelines require demonstrable expertise, authoritativeness, and trustworthiness. The site has no About page, no team bio, and no credentials section.

**What to add:**
- Who founded the business and their background
- What paralegal certification or training they hold
- Expanded Oregon Bar association disclaimer (currently only in footer)
- Any professional memberships (NALA, Oregon Paralegal Association, etc.)

### H6 — Reveal Animation Hides LCP Element
**Impact:** The hero section uses `class="reveal"` which sets `opacity: 0` initially. The `<h1>` inside the hero is the page's Largest Contentful Paint candidate. The initial hidden state can delay LCP measurement and confuse Google's CWV tooling.

The `.reveal` rule in `src/styles/main.css:177`:
```css
.reveal {
  opacity: 0;
  transform: translateY(20px);
  transition: opacity 0.65s ease, transform 0.65s ease;
}
```

**Fix:** The hero is always above the fold — it does not need a reveal animation. Remove `reveal` from the hero section, or at minimum from the `hero__copy` div that contains the H1.

---

### H7 — /intake/ Page Should Be Noindexed
**Impact:** The intake form is an internal operational form, not a public SEO page. It should not appear in search results.

**Fix in** `src/pages/intake.astro` — add `noindex` prop support to the Layout component:

`src/layouts/Layout.astro`:
```astro
const { title, description = '...', noindex = false } = Astro.props;
// In <head>:
{noindex && <meta name="robots" content="noindex, nofollow" />}
```

`src/pages/intake.astro`:
```astro
<Layout title="Client Intake Form" noindex={true}>
```

### M2 — Logo `<img>` Missing `width` Attribute
`src/layouts/Layout.astro:49`:
```html
<img src=... alt="A1 Quality Paralegal" loading="eager" height="58" />
```
Without both `width` and `height`, the browser cannot reserve space before the image loads, causing Cumulative Layout Shift (CLS).

**Fix:** Add the width based on the actual logo aspect ratio. If the logo is roughly 2:1 (58px tall):
```html
<img ... height="58" width="116" />
```
Apply the same fix to the footer logo instance.

---

### M3 — Skip Link Defined in CSS But Missing from HTML
`src/styles/main.css:54` has `.skip-link` styles but `src/layouts/Layout.astro` contains no skip link element. This breaks keyboard accessibility.

**Fix in** `src/layouts/Layout.astro` — add inside `<body>` before `<header>`:
```html
<a class="skip-link" href="#main">Skip to main content</a>
```

### M4 — Booking Page Google Calendar iframe Not Lazy-Loaded

## Long-Term Opportunities

**Content strategy gap is the biggest long-term limiter.** The site has zero blog or educational content. For "paralegal services Grants Pass OR", "will preparation Oregon", "power of attorney Josephine County" — there is real local search volume and almost no direct competition for informational queries. Publishing 8–10 helpful guides about Oregon estate planning basics would likely dominate local organic search within 3–6 months.

**Google Business Profile** — Not referenced anywhere on the site. If A1 Quality Paralegal has a GBP listing, it should be linked from the site and actively managed (photos, posts, review responses). The local 3-pack is far more visible than organic results for most local queries.
